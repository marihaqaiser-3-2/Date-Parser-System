📅 Date Input and Parsing Using Structure (C++)

📖 Overview

This C++ project demonstrates how to use a structure ("struct") to store date information and how to parse a date entered in the DD/MM/YYYY format using the "scanf()" function. The program separates the day, month, and year components and displays them individually.

---

✨ Features

- Stores date information using a structure.
- Accepts date input in "DD/MM/YYYY" format.
- Uses formatted input with "scanf()" to automatically separate values.
- Displays day, month, and year individually.
- Beginner-friendly example of structures and formatted input handling.

---

🏗️ Structure Details

Structure: "inputDate"

Data Member| Description
"day"| Stores the day value
"month"| Stores the month value
"year"| Stores the year value

---

▶️ Sample Output

Enter Date (DD/MM/YYYY): 12/06/2026
Day: 12
Month: 6
Year: 2026
